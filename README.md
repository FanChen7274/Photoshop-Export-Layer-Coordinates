# Photoshop-Export-Layer-Coordinates  

Photoshop 的 JSX 脚本，导出图层 XY 坐标到 txt文件  

1、安装  
(1)把jsx文件放到ps安装目录下的Presets\Scripts    
例如  D:\Photoshop CC2019\Adobe Photoshop CC 2019\Presets\Scripts    
(2)重启ps  
  
2、使用方法  
 (1)打开PS，打开需要导出坐标的PSD文件   
 (2)文件 > 脚本 > Export Layer Coordinates > 点击“选择文件夹”  
  
3、注意事项  
  脚本读取图形元素的坐标以自身左上角为参考点    
  坐标系统以画布左上角为原点 (0,0)，x 轴向右递增，y 轴向下递增
